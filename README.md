第一层：软件本身的登录密码，每次打开都要验证。

第二层：每个隐藏的文件或文件夹、每个加密保险箱，可以单独设置一个打开密码。

两道门，两把钥匙。就算别人知道你用了这个软件，没有第二层密码还是进不去。

### U 盘整体隐藏
